# 数组

## 数组的定义

由**相同类型**的元素的集合，**如果由不同类型的元素组成的集合成为元祖**

## 数组操作

### 新增

``` js
arr[index] = ele
```

### 修改

``` js
arr[index] = ele
```


### 删除

``` js
delete arr[index]
```

### 查找

``` js
arr[index]
```
