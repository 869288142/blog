# 闭包
函数中定义的函数

## 本质
保存外部函数 AO 对象即作用链上一层 AO 对象

## 副作用   
不取消 AO 引用容易引起内存泄漏

## 作用
缓存

模仿块级作用域

```javascript
(function() {
    //块级作用域
})();
```

私有变量

模拟面向对象中的类

```javascript
function Person() {
   

    return {
         var name = "default";
        getName: function() {
            return name;
        },
        setName: function(newName) {
            name = newName;
        }
    };
}
```

循环和闭包
模块化
