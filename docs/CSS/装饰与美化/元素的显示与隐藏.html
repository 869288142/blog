# 元素的隐藏和显示

# 隐藏实践

  1. 利用script标签隐藏或者textarea，元素 不加载
<script type="text/html">
  <img src="1.jpg">
</script>
  1. 元素加载，DOM可访问 display none
  2. 占据空间 visibility: hidden
  3. 键盘可访问 clip裁剪
  4. 占据空间， 键盘可访问 z-index
  5. 可以点击 透明度
